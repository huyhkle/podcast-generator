# Podcast Generator

Tự động tạo script podcast tin tức tiếng Việt mỗi ngày, powered by Claude AI.

## Cách dùng

1. Mở link web app trên điện thoại hoặc máy tính
2. Chọn loại podcast muốn generate
3. Nhấn "Generate Script Hôm Nay"
4. Copy script → paste vào NotebookLM → Audio Overview

## Setup

Cần set environment variable `ANTHROPIC_API_KEY` trên Vercel.
